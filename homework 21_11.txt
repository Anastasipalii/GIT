 1. Для копирования репозитория с GitHub на свой компьютер, нужно проверить установлен ли Git. Затем выполните команду git clone с URL репозитория.  git clone https://github.com/пользователь/репозиторий.git

2. Чтобы создать Git репозиторий из локального каталога, нужно перейти в каталог и выполнить git init. Чтобы отправить его на GitHub, нужно создайть новый репозиторий на сайте GitHub и выполните инструкции "…or push an existing repository from the command line", добавив удаленный репозиторий с помощью git remote add origin https://github.com/пользователь/репозиторий.git и затем git push -u origin master.

3. Для получения изменений с GitHub на свой локальный компьютер нужно использовать команду git pull.

4. Команда git status показывает состояние  рабочего каталога, git log выводит историю коммитов, а git remote -v отображает адреса удаленных репозиториев.
